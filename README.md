🦖 MongoDB Dinosaur Operations – Parth-ctrl490
Overview 📝

This repository contains real MongoDB Shell commands for a dinosaur-themed database. It’s designed to help beginners and intermediate learners practice CRUD operations, aggregation, indexing, schema validation, and joins using MongoDB. 🚀

✨ Features

🔹 CRUD Operations – Insert, update, delete, and view documents

🔹 Text Index & Search – Full-text search with indexing

🔹 Aggregation – Use $group and other aggregation pipelines

🔹 Schema Validation – Ensure data integrity with $jsonSchema

🔹 Lookup (Join) – Join collections using $lookup

🔹 Sorting & Pagination – Sort, limit, and skip results

🔹 Clean, beginner-friendly examples
